


# Linux — Tools

> The power tools — grep, awk, sed, find, pipes, redirects, and bash scripting. These are the glue that makes everything else work.

---

## Pipes & Redirection

```bash
# pipes — pass output of one command to another
command1 | command2
ls -la | grep ".sh"
cat file.txt | sort | uniq

# redirection
command > file.txt      # write stdout to file (overwrite)
command >> file.txt     # append stdout to file
command 2> error.txt    # write stderr to file
command 2>&1            # redirect stderr to stdout
command > out.txt 2>&1  # both stdout and stderr to file
command < file.txt      # use file as stdin
command &> file.txt     # shorthand for both stdout + stderr
```

### /dev/null

```bash
command 2>/dev/null     # discard errors
command > /dev/null     # discard output
command &>/dev/null     # discard everything
```

## grep

Search for patterns in text.

```bash
grep "pattern" file.txt             # basic search
grep -i "pattern" file.txt          # case insensitive
grep -r "pattern" dir/              # recursive
grep -n "pattern" file.txt          # show line numbers
grep -v "pattern" file.txt          # invert — lines NOT matching
grep -l "pattern" *.txt             # only show filenames
grep -c "pattern" file.txt          # count matches
grep -A 3 "pattern" file.txt        # 3 lines after match
grep -B 3 "pattern" file.txt        # 3 lines before match
grep -E "pat1|pat2" file.txt        # extended regex — OR
grep -o "pattern" file.txt          # only print match, not whole line
```

```bash
# useful combinations
ps aux | grep python                # find python processes
cat /etc/passwd | grep "/bin/bash"  # users with bash shell
grep -r "password" /var/www/ 2>/dev/null  # find passwords in web files
```

## find

Find files and directories.

```bash
find / -name "file.txt"             # find by name
find . -name "*.py"                 # find by extension
find / -type f -name "*.conf"       # files only
find / -type d -name "logs"         # directories only
find / -user root                   # files owned by root
find / -size +100M                  # files over 100MB
find / -size -1k                    # files under 1KB
find / -mtime -1                    # modified in last 24h
find / -mtime +30                   # modified more than 30 days ago
find / -perm -u=s                   # SUID files
find / -perm -o+w                   # world-writable files
find . -empty                       # empty files/dirs
```

```bash
# execute command on results
find . -name "*.log" -exec rm {} \;         # delete all .log files
find . -name "*.py" -exec grep "TODO" {} \; # grep inside found files
find . -name "*.sh" -exec chmod +x {} \;    # make scripts executable
```

## sed — Stream Editor

Find and replace text in files or streams.

```bash
sed 's/old/new/' file.txt           # replace first match per line
sed 's/old/new/g' file.txt          # replace all matches
sed 's/old/new/gi' file.txt         # case insensitive
sed -i 's/old/new/g' file.txt       # edit file in place
sed -n '5,10p' file.txt             # print lines 5-10
sed '5,10d' file.txt                # delete lines 5-10
sed '/pattern/d' file.txt           # delete lines matching pattern
sed 's/^/  /' file.txt              # add 2 spaces to start of each line
sed 's/ *$//' file.txt              # remove trailing whitespace
```

```bash
# useful combinations
cat file.txt | sed 's/foo/bar/g'                    # pipe usage
sed -i 's/localhost/10.0.0.1/g' config.php          # update config
ps aux | sed 's/  */ /g'                            # collapse whitespace
```

## awk

Process and extract columns from structured text.

```bash
awk '{print $1}' file.txt           # print first column
awk '{print $1, $3}' file.txt       # print columns 1 and 3
awk -F: '{print $1}' /etc/passwd    # custom delimiter (:)
awk 'NR==5' file.txt                # print line 5
awk 'NR>=5 && NR<=10' file.txt      # print lines 5-10
awk '/pattern/' file.txt            # print matching lines
awk '{sum += $1} END {print sum}'   # sum a column
awk '{print NR, $0}' file.txt       # add line numbers
awk 'length($0) > 80' file.txt      # lines longer than 80 chars
```

```bash
# useful combinations
cat /etc/passwd | awk -F: '{print $1}'          # list all usernames
ps aux | awk '{print $1, $2, $11}'              # user, PID, command
df -h | awk '{print $1, $5}'                    # filesystem + use%
```

## cut

Extract columns by delimiter or character position.

```bash
cut -d: -f1 /etc/passwd             # field 1, delimiter :
cut -d, -f1,3 file.csv              # fields 1 and 3, comma delimited
cut -c1-10 file.txt                 # characters 1-10
```

## sort & uniq

```bash
sort file.txt                       # alphabetical sort
sort -n file.txt                    # numerical sort
sort -r file.txt                    # reverse sort
sort -k2 file.txt                   # sort by column 2
sort -u file.txt                    # sort + remove duplicates

uniq file.txt                       # remove consecutive duplicates
uniq -c file.txt                    # count occurrences
uniq -d file.txt                    # only show duplicates
```

```bash
# count most frequent items
sort file.txt | uniq -c | sort -rn | head -10
```

## tr — Translate Characters

```bash
tr 'a-z' 'A-Z' < file.txt          # lowercase to uppercase
tr -d '\n' < file.txt               # remove newlines
tr -d ' ' < file.txt                # remove spaces
tr ':' '\n' < file.txt              # replace : with newline
tr -s ' ' < file.txt                # squeeze multiple spaces to one
```

---

## xargs

Pass output as arguments to another command.

```bash
find . -name "*.txt" | xargs rm             # delete found files
cat urls.txt | xargs curl -O               # download list of URLs
find . -name "*.py" | xargs grep "import"  # grep in found files
echo "1 2 3" | xargs -n1 echo             # one arg per line
```

---

## tee

Write to file AND stdout simultaneously.

```bash
command | tee file.txt              # output to screen + file
command | tee -a file.txt           # append to file
command | tee file.txt | grep error # pipe onwards after saving
```

## Bash Scripting

### Structure

```bash
#!/bin/bash
# comment

# variables
NAME="world"
echo "Hello $NAME"

# command output to variable
FILES=$(ls -la)
DATE=$(date +%Y-%m-%d)
```

### Conditionals

```bash
if [ condition ]; then
    commands
elif [ condition ]; then
    commands
else
    commands
fi
```

```bash
# common conditions
[ -f file ]         # file exists
[ -d dir ]          # directory exists
[ -z "$var" ]       # variable is empty
[ -n "$var" ]       # variable is not empty
[ "$a" == "$b" ]    # string equality
[ $n -eq 5 ]        # numeric equal
[ $n -gt 5 ]        # greater than
[ $n -lt 5 ]        # less than
```

### Loops

```bash
# for loop
for i in 1 2 3 4 5; do
    echo $i
done

for file in *.txt; do
    echo "Processing $file"
done

for i in $(seq 1 10); do
    echo $i
done

# while loop
while [ condition ]; do
    commands
done

# read lines from file
while IFS= read -r line; do
    echo "$line"
done < file.txt
```

### Functions

```bash
greet() {
    local name=$1       # local variable
    echo "Hello $name"
}

greet "Alice"
```

### Arguments

```bash
$0          # script name
$1, $2      # arguments
$#          # number of arguments
$@          # all arguments
$?          # exit code of last command
$$          # current PID
```

### Error Handling

```bash
set -e              # exit on error
set -u              # exit on undefined variable
set -x              # print each command (debug)

command || exit 1   # exit if command fails
command || echo "failed" && exit 1
```

## Useful One-liners

```bash
# find and replace in all files recursively
grep -rl "old_text" . | xargs sed -i 's/old_text/new_text/g'

# count lines in all Python files
find . -name "*.py" | xargs wc -l | tail -1

# watch a log file for errors
tail -f /var/log/syslog | grep -i error

# list top 10 largest files
find . -type f | xargs du -sh | sort -rh | head -10

# extract IPs from a file
grep -oE '[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}' file.txt

# remove blank lines from file
sed '/^$/d' file.txt

# show only unique lines between two files
comm -23 <(sort file1.txt) <(sort file2.txt)

# run command on remote server
ssh user@host "command"

# monitor a directory for changes
watch -n 1 ls -la /tmp/
```
