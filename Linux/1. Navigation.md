



# Linux — Navigation

---

## Filesystem Structure

```
/
├── bin/       — essential binaries (ls, cp, mv)
├── etc/       — system config files
├── home/      — user home directories (/home/username)
├── var/       — logs, caches, variable data
├── tmp/       — temporary files (cleared on reboot)
├── usr/       — user programs and libraries
├── opt/       — optional/third-party software
├── proc/      — virtual fs — running processes info
└── dev/       — device files (disks, terminals)
```

|Path|Shorthand|
|---|---|
|Root|`/`|
|Home directory|`~`|
|Current directory|`.`|
|Parent directory|`..`|

---

## Moving Around

|Command|What it does|
|---|---|
|`pwd`|Print current directory|
|`cd /path/to/dir`|Go to absolute path|
|`cd ~/projects`|Go to path relative to home|
|`cd ..`|Go up one level|
|`cd -`|Go to previous directory|
|`ls`|List files|
|`ls -la`|List all files with permissions + hidden|
|`ls -lh`|Human-readable file sizes|
|`tree`|Visual directory tree|

---

## File Operations

|Command|What it does|
|---|---|
|`cp file dest`|Copy file|
|`cp -r dir dest`|Copy directory recursively|
|`mv file dest`|Move or rename|
|`rm file`|Delete file|
|`rm -rf dir`|Delete directory recursively — **no undo**|
|`mkdir dir`|Create directory|
|`mkdir -p a/b/c`|Create nested directories|
|`touch file`|Create empty file / update timestamp|
|`cat file`|Print file contents|
|`less file`|Scroll through file|
|`head -n 20 file`|First 20 lines|
|`tail -n 20 file`|Last 20 lines|
|`tail -f file`|Follow file live (logs)|

---

## Paths

|Type|Example|Meaning|
|---|---|---|
|Absolute|`/home/user/projects`|From root — always works|
|Relative|`./projects` or `../other`|From current location|
|Home-relative|`~/projects`|From home dir|

## Symlinks

```bash
ln -s /path/to/original linkname   # create symlink
ls -la                              # symlinks show as link -> target
readlink -f linkname                # resolve full path
```

|Command|What it does|
|---|---|
|`df -h`|Disk usage per filesystem|
|`du -sh dir`|Size of a directory|
|`du -sh *`|Size of each item in current dir|

## Finding Files

|Command|What it does|
|---|---|
|`find . -name "*.py"`|Find by name|
|`find . -type f -size +100M`|Find files over 100MB|
|`find . -mtime -1`|Modified in last 24h|
|`which python`|Location of a binary|
|`whereis python`|All locations of a binary|

Full `find` power in `Tools.md`.

## Wildcards & Globbing

|Pattern|Matches|
|---|---|
|`*`|Any characters|
|`?`|Single character|
|`[abc]`|a, b, or c|
|`{py,js}`|py or js|

```bash
ls *.md          # all markdown files
rm log_?.txt     # log_1.txt, log_2.txt ...
cp {a,b}.py dst/ # copy a.py and b.py
```

## History & Shortcuts

|Shortcut|Action|
|---|---|
|`↑ / ↓`|Previous / next command|
|`Ctrl+R`|Search command history|
|`!!`|Repeat last command|
|`!$`|Last argument of previous command|
|`Ctrl+C`|Kill current command|
|`Ctrl+L`|Clear screen|
|`Tab`|Autocomplete|

```bash
sudo !!          # repeat last command with sudo
cd !$            # cd into last argument
```
