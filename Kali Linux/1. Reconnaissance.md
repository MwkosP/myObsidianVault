


# Kali — Reconnaissance

> Recon = gathering information before touching the target. Passive = no direct contact. Active = direct contact.

---

## Methodology

```
Passive Recon          → no packets to target
    ↓
OSINT / DNS / Whois
    ↓
Active Recon           → packets hit target
    ↓
Port scan → Service enum → Vuln scan
```

---

## Passive Recon

### whois

```bash
whois example.com           # registrar, owner, nameservers
whois 1.2.3.4               # IP ownership / ASN
```

### DNS Enumeration

```bash
nslookup example.com                    # basic lookup
dig example.com ANY                     # all records
dig example.com MX                      # mail servers
dig example.com NS                      # nameservers
dig axfr @ns1.example.com example.com  # zone transfer (if misconfigured)
dnsrecon -d example.com                 # automated DNS recon
dnsenum example.com                     # enum + zone transfer attempt
```

|DNS Record|Meaning|
|---|---|
|A|IPv4 address|
|AAAA|IPv6 address|
|MX|Mail server|
|NS|Nameserver|
|TXT|SPF, DKIM, misc info|
|CNAME|Alias to another domain|
|SOA|Zone authority info|

### Subdomain Discovery

```bash
sublist3r -d example.com               # passive subdomain enum
amass enum -passive -d example.com     # OSINT subdomain discovery
assetfinder example.com                # fast subdomain finder
```

## OSINT

### theHarvester

```bash
theHarvester -d example.com -b all         # emails, IPs, subdomains
theHarvester -d example.com -b google      # google only
theHarvester -d example.com -b linkedin    # linkedin
```

### Google Dorks

```bash
site:example.com                    # all indexed pages
site:example.com filetype:pdf       # PDFs only
site:example.com inurl:admin        # admin pages
intext:"index of /" site:example.com # open directories
"example.com" filetype:sql          # exposed databases
intitle:"login" site:example.com    # login pages
```

### Shodan

```bash
# at shodan.io or CLI
shodan host 1.2.3.4                 # host info
shodan search "apache" country:GR   # search by banner
shodan search org:"Target Corp"     # by org name
```

|Shodan Filter|Use|
|---|---|
|`hostname:example.com`|By hostname|
|`port:22`|By port|
|`org:"name"`|By organization|
|`country:US`|By country|
|`product:nginx`|By software|

## Active Recon

### nmap — Host Discovery

```bash
nmap -sn 192.168.1.0/24             # ping sweep — find live hosts
nmap -sn 192.168.1.0/24 --open      # only show up hosts
nmap -PR 192.168.1.0/24             # ARP scan (local network)
```

### nmap — Port Scanning

```bash
nmap 10.0.0.1                       # top 1000 ports (TCP)
nmap -p- 10.0.0.1                   # all 65535 ports
nmap -p 22,80,443 10.0.0.1          # specific ports
nmap -sU 10.0.0.1                   # UDP scan
nmap -sS 10.0.0.1                   # SYN scan (stealthy, needs root)
nmap -sV 10.0.0.1                   # service version detection
nmap -O 10.0.0.1                    # OS detection
nmap -A 10.0.0.1                    # aggressive: OS + version + scripts
nmap -T4 10.0.0.1                   # faster timing
```

|nmap Flag|Meaning|
|---|---|
|`-sS`|SYN scan (half-open, stealthy)|
|`-sT`|Full TCP connect scan|
|`-sU`|UDP scan|
|`-sV`|Version detection|
|`-sC`|Default scripts|
|`-O`|OS detection|
|`-A`|All of the above|
|`-p-`|All ports|
|`-T1-5`|Timing (1=slow/stealthy, 5=fast)|
|`--open`|Only show open ports|
|`-oN file`|Save output to file|
|`-oX file`|Save as XML|

### nmap — Scripts (NSE)

```bash
nmap --script=default 10.0.0.1              # default scripts
nmap --script=vuln 10.0.0.1                 # vuln detection
nmap --script=http-enum 10.0.0.1            # web enumeration
nmap --script=smb-vuln* 10.0.0.1            # SMB vulns
nmap -p 80 --script=http-title 10.0.0.1     # grab page titles
```

## Saving & Organizing Output

```bash
nmap -A -p- 10.0.0.1 -oA scan_results   # save as .nmap, .xml, .gnmap
cat scan_results.nmap                    # read normal output
```

## Common Recon Workflow

```bash
# 1. find live hosts
nmap -sn 10.0.0.0/24 -oN hosts.txt

# 2. full port scan on target
nmap -p- -T4 10.0.0.1 -oN allports.txt

# 3. version + script scan on open ports
nmap -sV -sC -p 22,80,443 10.0.0.1 -oN detailed.txt

# 4. DNS recon
dnsrecon -d example.com

# 5. subdomain discovery
sublist3r -d example.com -o subdomains.txt

# 6. OSINT
theHarvester -d example.com -b all -f results.html
```
